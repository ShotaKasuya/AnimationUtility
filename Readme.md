## Animator Controller Bind Generator

指定フォルダに`AnimationController`の`Parameter`と`State`のハッシュ値とenumの対応を自動生成する。
生成は`Tools/Animator Controller Binding Generator`からできます。
`AssetPostprocessor`を利用して生成を自動化したらロードが遅くなった気がしたため。
<img width="942" height="253" alt="image" src="https://github.com/user-attachments/assets/680f1701-1ac2-4fdf-8e36-480be86a775e" />
<img width="1112" height="373" alt="image" src="https://github.com/user-attachments/assets/4be94c5a-cdd6-4711-9dea-25d8490e7e23" />

```csharp
// Auto-generated by AnimatorBindingGenerator
using UnityEngine;
using System.Collections.Generic;

public static class ControllerBind
{
    public enum Parameter
    {
        Up,
        Down,
        Left,
        Right,
    }

    private static readonly Dictionary<Parameter, int> ParameterHashes = new Dictionary<Parameter, int>
    {
        { Parameter.Up, Animator.StringToHash("Up") },
        { Parameter.Down, Animator.StringToHash("Down") },
        { Parameter.Left, Animator.StringToHash("Left") },
        { Parameter.Right, Animator.StringToHash("Right") },
    };

    public static int ToHash(this Parameter parameter)
    {
        return ParameterHashes[parameter];
    }

    public enum State
    {
        Default,
        LeftUp,
        RightDown,
    }

    private static readonly Dictionary<State, int> StateHashes = new Dictionary<State, int>
    {
        { State.Default, Animator.StringToHash("Default") },
        { State.LeftUp, Animator.StringToHash("LeftUp") },
        { State.RightDown, Animator.StringToHash("RightDown") },
    };

    public static int ToHash(this State state)
    {
        return StateHashes[state];
    }
}

```
